## Run Locally

Clone the project

```bash
  git clone https://github.com/0xhylia/hylia.dev
```

Go to the project directory

```bash
  cd hylia.dev
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
    pnpm run dev
```
