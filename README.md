
| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |

## Run Locally

Clone the project

```bash
  git clone https://github.com/akenolol/akenolol.github.io
```

Go to the project directory

```bash
  cd akenolol.github.io
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
