## Run Locally

Clone the project

```bash
  git clone https://github.com/0xhylia/hylia.dev
```

Go to the project directory

```bash
  cd hylia.dev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
